What is a Contract Definition?

A Contract Definition is a template that defines the terms and conditions under which data assets can be shared. It combines asset selection criteria, access policies, and contract policies to create a complete data sharing agreement framework.

Unique identifier for your contract definition
Human-readable description of this contract's purpose
Specify which assets this contract covers
Define who can access the data
Comma-separated list of BPNs
Define how the data can be used
👁️ Live Preview & API Integration

How Contract Definitions Work

Contract definitions are the foundation of EDC data sharing. They create reusable templates that automatically apply business rules when partners request data access.
